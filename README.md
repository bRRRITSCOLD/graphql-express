Node.js/Express CRUD backend using GraphQL and JSON-Server

### Version
1.0.0

## Usage

1) Install Dependencies

```bash
$ npm install
```

or 

```bash
yarn install
```

2) JSON-Server (Port 3000)

```bash
$ npm run json:server
```

or 

```bash
yarn run json:server
```

3) Server (Port 4000)

```bash
$ npm run dev:server
```
or 

```bash
yarn run dev:server
```

Go to http://localhost:4000/graphql
